Introduction to Python

Python is a high-level, interpreted programming language known for its simplicity and readability. Created by Guido van Rossum in 1991, Python is widely used in web development, data science, artificial intelligence, automation, and more.

Why Learn Python?

  • Easy to Learn – Simple syntax similar to English.
  • Versatile – Can be used for web apps, data science, AI, etc.
  • Huge Community Support – Many resources and libraries.
  • Career Opportunities – High demand in various industries.
  • Cross-Platform – Runs on Windows, macOS, and Linux.

Installing Python

  1. Download Python from python.org.
  2. Install it and check by running:
    3. python --version
  3. Use IDLE or VS Code/PyCharm for writing Python programs.

Variables and Data Types in Python

In Python, a variable is like a container that stores data. You can assign a value to a variable using the = sign. For example, name = "Alice" stores the text "Alice" in the variable name. Python has different data types to handle various kinds of data:

Integer: Whole numbers e.g., x = 10

Float: Decimal numbers e.g., price = 10.5

String: Text e.g., name = "Alice"

Boolean: True or False values e.g, is_raining = False

Python is dynamically typed, meaning you don’t need to specify the data type—it’s assigned automatically. You can check a variable’s type using the type() function, like print(type(name))

name = "Shifa"
age = 20
is_student = True

Input and Output

name = input("Enter your name: ")
print("Hello,", name)

Control Statements in Python

Control statements help us control the flow of a program based on conditions and loops. There are two main types:

  • If-Else Statements – Used to make decisions
  • Loops – Used to repeat a block of code

1️⃣ If-Else Statement

An if-else statement checks a condition and decides what to do based on whether the condition is True or False.

age = int(input("Enter your age: "))
if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

How it works?

  • The program asks the user to enter their age.
  • The if condition checks if the age is 18 or more.
  • If True, it prints "You are an adult."
  • If False, it prints "You are a minor."

💡 Use case: If-Else is useful when making decisions, like checking if a user is old enough to vote.

2️⃣ Loops in Python

Loops are used to repeat a block of code multiple times. Python has two main loops:

🔹 For Loop

A for loop is used when we know how many times we want to repeat something.

for i in range(5):
    print(i)

How it works?

  • range(5) generates numbers 0 to 4 (not including 5).
  • The loop runs 5 times, printing numbers 0 to 4 one by one.

💡 Use case: Use a for loop when you want to repeat a task a set number of times, like printing numbers or looping through a list.

🔹 While Loop

A while loop runs as long as a condition is True.

x = 0
while x < 5:
    print(x)
    x += 1

How it works?

  • The loop starts with x = 0
  • The condition x < 5 is checked before each loop run.
  • Inside the loop, x is printed, and then increased by 1 x += 1
  • The loop stops when x becomes 5

💡 Use case: Use a while loop when you don't know how many times the loop should run, like waiting for user input.

Summary

  • ✅ If-Else → Makes decisions e.g., checking age.
  • ✅ For Loop → Repeats a block a fixed number of times.
  • ✅ While Loop → Repeats until a condition is false.

Functions in Pyton

A function in Python is a reusable block of code that performs a specific task. Instead of writing the same code again and again, we can use functions to make our programs shorter and easier to understand.

📌 Example Code:

def greet(name):
                              return "Hello, " + name
                          
                          print(greet("Shifa"))

🔹 How This Code Works?

  • The keyword def is used to define a function
  • The function greet(name) takes one input (name)
  • Inside the function, it returns "Hello, " + name which means it adds the name to "Hello, "
  • We call the function using print(greet("Shifa"))
  • This prints "Hello, Shifa" on the screen.

✅ Why Use Functions?

  • ⭐ Reusability: We can call the function many times instead of writing the same code again.
  • ⭐ Better Readability: Code looks cleaner and is easier to understand.
  • ⭐ Easier to Fix Errors: If something goes wrong, we only need to change the function instead of the entire program.

🚀 Example: Calling the Function Multiple Times

print(greet("Alice"))
                          print(greet("Bob"))
                          print(greet("Charlie"))

Output:

Hello, Alice
                          Hello, Bob
                          Hello, Charlie

This shows how functions help us avoid repeating code! 🎯

Lists and Tuples in Python

Python provides different ways to store multiple values in a single variable. Two common types are:

  • Lists - A collection of items that can be changed (mutable).
  • Tuples - A collection of items that cannot be changed (immutable).

🔹 Lists

A list is like a container that holds multiple values. You can add, remove, or change items in a list.

fruits = ["Apple", "Banana", "Cherry"]
fruits.append("Mango")
print(fruits)

How it works?

  • fruits is a list containing Apple, Banana, Cherry.
  • append("Mango") adds Mango to the list.
  • After running this code, the list becomes ["Apple", "Banana", "Cherry", "Mango"].

🔹 Tuples

A tuple is similar to a list, but you cannot change its values after creating it.

days = ("Monday", "Tuesday", "Wednesday")
print(days[0])

How it works?

  • days is a tuple containing Monday, Tuesday, and Wednesday.
  • print(days[0]) prints the first value, which is Monday.
  • Unlike lists, you cannot add or remove items from a tuple.

Dictionaries in Python

A dictionary stores data in key-value pairs. It is useful for organizing related information.

person = {"name": "Shifa", "age": 20}
print(person["name"])  # Output: Shifa

How it works?

  • person is a dictionary with two keys: name and age.
  • person["name"] returns Shifa, which is stored under the name key.
  • Dictionaries allow storing different types of information in an organized way.

✅ Why Use These Data Types?

  • Lists → When you need a collection of items that can change.
  • Tuples → When you need a collection that should stay the same.
  • Dictionaries → When you need to store related data using key-value pairs.

These are some of the most useful data types in Python for organizing data efficiently!

Object-Oriented Programming (OOP) in Python

Object-Oriented Programming (OOP) is a way to organize code by creating objects. An object is like a real-world item, such as a car or a person. These objects have attributes (characteristics) and methods (actions).

🔹 What are Classes and Objects?

  • Class → A template used to create objects.
  • Object → A specific example created from a class.

📌 Example Code:

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        print("Hello, my name is " + self.name)

p = Person("Shifa", 20)
p.greet()

🔹 Explanation:

  • class Person creates a blueprint for a person.
  • __init__ is a special function that runs when we create a new person.
  • self.name and self.age store the person’s name and age.
  • The greet() function prints a message with the person’s name.
  • p = Person("Shifa", 20) creates a person named Shifa, who is 20 years old.
  • p.greet() calls the function and prints "Hello, my name is Shifa".

✅ Why Use OOP?

  • Easy to Understand → Code is organized into objects, like real-world things.
  • Reusable → You can create multiple objects from the same class.
  • Less Repetition → Functions inside classes make code shorter.

🚀 Creating Multiple Objects

p1 = Person("Alice", 25)
p2 = Person("Bob", 30)

p1.greet()  # Output: Hello, my name is Alice
p2.greet()  # Output: Hello, my name is Bob

OOP is widely used in programming, from making apps to building websites. Learning OOP makes Python easier to use!

File Handling in Python

File handling allows us to read and write data to files. This is useful for saving and retrieving information.

🔹 Reading a File

The following code reads the content of a file and prints it:

with open("file.txt", "r") as file:
    print(file.read())

Explanation:

  • open("file.txt", "r") opens the file in read mode.
  • file.read() reads the file’s content.
  • The with statement ensures the file is closed automatically.

🔹 Writing to a File

The following code writes text into a file:

with open("file.txt", "w") as file:
    file.write("Hello, Python!")

Explanation:

  • open("file.txt", "w") opens the file in write mode.
  • file.write("Hello, Python!") writes text into the file.
  • If the file doesn’t exist, Python will create it.

Exception Handling in Python

Exceptions are errors that occur during program execution. Python allows us to handle these errors to prevent crashes.

📌 Example Code:

try:
    x = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero!")

Explanation:

  • The try block contains code that might cause an error.
  • The except block catches the error and prevents the program from stopping.
  • Here, dividing by zero is not allowed, so the program prints "Cannot divide by zero!".
